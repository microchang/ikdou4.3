一期需求

  1. 完成基本模块搭建！
  2. 完成音乐，电影模块开发
  3. 完善服务端，客户端界面

最低系统需求

  • 必须是基于node.js的express4.x版，推荐express4.2
  • 命名必须符合路由规则，否则无法匹配！
  • 推荐node.js版本0.10.26以上版本
  • 基于html5，css3，angularjs1.3以上版本！
  • 推荐jQuery版本2.1以上版本

IDE推荐

  • 推荐使用webstorm 8.x版，是目前对angularjs，node.js支持最好的IDE
